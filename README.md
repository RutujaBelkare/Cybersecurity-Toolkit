#  Cyber Security Toolkit

A Python-based Cyber Security Toolkit that integrates multiple security functionalities into a single application. This project demonstrates basic cybersecurity concepts like encryption, password analysis, keylogging, and network packet inspection.

---



-  Caesar Cipher (Encryption & Decryption)
-  Image Encryption using Pixel Manipulation
-  Password Strength Checker
-  Keylogger (Educational Use Only)
-  Network Packet Analyzer (Packet Sniffer)

---

##  Technologies Used

- Python
- Pillow (Image Processing)
- Pynput (Keylogger)
- Scapy (Packet Sniffing)
